🔒 What is Blockchain?

A blockchain is a distributed ledger where each block contains:

  • ✓ Cryptographic hash of previous block
  • ✓ Timestamp for verification
  • ✓ Transaction data
  • ✓ Proof-of-Work (nonce)

⚙️ How It Works

  • • Each block is cryptographically linked
  • • Mining requires computational work
  • • Changing data breaks the chain
  • • Hashes ensure immutability
