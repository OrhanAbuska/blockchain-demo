# Interactive Blockchain Demo

An interactive, educational blockchain demonstration built with pure HTML/CSS/JavaScript. No backend required - runs entirely in your browser!

## Live Demo
[View Demo](https://orhanabuska.github.io/blockchain-demo)

## Features
- Add new blocks with custom transaction data
- Tamper with existing blocks to see chain break
- Mine blocks with adjustable difficulty
- Real-time blockchain visualization
- Export blockchain data as JSON
- Responsive design for all devices

## How It Works
1. Each block contains data, timestamp, and hash
2. Blocks are cryptographically linked using SHA-256
3. Mining requires finding a hash with leading zeros
4. Tampering breaks the chain integrity

## Tech Stack
- HTML5
- CSS3 (Grid, Flexbox, Animations)
- Vanilla JavaScript
- Web Crypto API for SHA-256 hashing

## Local Setup
1. Clone the repository
2. Open `index.html` in your browser
3. Start exploring blockchain concepts!
